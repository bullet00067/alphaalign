# AlphaAlign

## 🚀 互動式資產再平衡與監控平台 (Interactive Asset Rebalancing & Monitoring Platform)

AlphaAlign 是一個專為個人投資者設計的**動態資產監控與再平衡系統**。透過直覺的動態互動介面與精確的計算引擎，協助使用者消除手動計算權重與買賣單位的繁瑣流程，並長期追蹤資產成長趨勢。

---

## ✨ 核心功能 (Key Features)

### 1. 動態配置與互動調整介面
* **靈活配置卡片：** 自由新增資產類別（如：市值型、高股息、債券、現金），並綁定具體標的 (Ticker)。
* **待分配資金池機制：** 即時顯示配置總和，並提供 1% 步進調整，確保配置精確達成 100%。
* **防呆鎖定邏輯：** 自動驗證配置比例，未滿 100% 時禁用執行功能，避免操作失誤。

### 2. 試算與儲存之雙軌機制
* **僅單次試算 (Simulate Only)：** 快速模擬調整結果，不更動資料庫設定，適合短期戰術分析。
* **更新配置並產出報告 (Save Target & Rebalance)：** 永久儲存新權重，並產出完整的買賣執行清單。

### 3. 雙重驗證報價系統 (Dual-Verification)
為確保數據絕對精準，系統整合了雙重驗證機制：
* **主要來源：** 金融數據 API (yfinance)。
* **次要來源：** 非同步輕量級爬蟲 (Yahoo Finance 網頁抓取)。
* **比對邏輯：** 當 API 數據有顯著延遲或偏差時，自動選用最新網頁價格作為計算基準。

### 4. 智慧再平衡計算與交易成本估算
* **全方位淨值計算：** 整合現金、資產市值與預計加碼資金。
* **交易成本自動扣除：**
    * **台股：** 預扣券商手續費 (0.1425%) 與賣出證交稅 (0.3%)。
    * **美股：** 依據設定扣除海外券商固定手續費或複委託成本。
* **精確單位換算：** 台股自動處理張/股轉換；美股支援碎股計算（至小數點後兩位）。

### 5. 資產監控中心與歷史快照
* **直覺儀表板：** 圖表呈現「目前佔比 vs. 目標佔比」的偏離度。
* **成長曲線：** 自動紀錄歷史再平衡快照，繪製長線資產成長趨勢。

## 🛠 技術架構 (Tech Stack)

* **前端 (Frontend)：** React / Vue 3 + Tailwind CSS (部署於 Render Static Site)
* **後端 (Backend)：** Python (FastAPI) + Docker (部署於 Render Web Service)
* **資料庫 (Database)：** Managed PostgreSQL (Render)
* **數據抓取：** yfinance + Custom Async Scraper

---

## 📡 API 設計概要

* `GET /api/portfolio` - 獲取庫存與目標比例
* `POST /api/rebalance/calculate` - 單次試算再平衡建議
* `PUT /api/allocations/targets` - 儲存目標配置比例
